Neste repositório, apresento os scripts principais que utilizamos para coleta dos dados, filtrar as top 20 doenças, tratar dummy variables e plot das correlações. 
Na sequencia, encontramos os arquivos:
1 Importacao_ANS_INMET.r
2 top_20.r
3 dummies.r
4 corr.r

Este repositório também possui o arquivo "Relatorios TCC.pbix", da qual possui os gráficos apresentados no trabalho. Para conseguir abrir o arquivo, será necessário instalar a versão gratuita do Microsoft Powerbi encontrado no link https://powerbi.microsoft.com/pt-br/downloads/
